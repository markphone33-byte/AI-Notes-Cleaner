# AI Notes Cleaner

A Python application that organizes messy notes using the OpenAI API.


## Features
- Multi-line note input
- AI-generated summaries
- Terminal-based workflow
- Optional Zapier/Google Docs integration


## Technologies Used
- Python
- OpenAI API
- Git/GitHub
- Zapier


## How It Works
1. User pastes messy notes into terminal
2. Program sends notes to the OpenAI API
3. AI turns notes into organized sections
4. Output is displayed in terminal
5. Optional: Saves notes as a file or turn them into a Google Doc with Zapier


## Future Improvements
- GUI/Desktop app
- File export options
- Google Docs API integration


## How to Run
1. Install dependencies
    Run: pip3 install -r requirements.txt
2. Set up your OPENAI_API_KEY environment variable
    Run: export OPENAI_API_KEY="your_api_key_here"
3. Run the program
    Run: python3 ImproveNotes.py
